Настройка GIT  - выолняется один раз, после установки на компьютер

git config  --global user.email  roman@com.ua


git config  --global user.name  yuliya
git config  --global user.email yulchik.sushko1712@gmail.com  

git config  --global color.ui  true
git config --global credential.helper cache
git config --global credential.helper 'cache --timeout=3600'

:wq  - сохранить и выйти из редактора
:q!  - выйти из редактора без созранения
