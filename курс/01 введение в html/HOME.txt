При выполнении задания обращать внимание на
- на те сообщения, который выводит git в консоль.
  Надо проанализировать каждое сообщение
- содержание описания коммита !


1. В каталоге проектов создать каталог Home_1 и 
скопировать в него файл .gitingnore из каталога 01_html

2. В созданном каталоге создать файл index.html  и добавить в него 1 заголовок h1 и 2 параграфа текста  (использовать генератор lorem ipsum...)

3. В созданном каталоге создать репозиторий git, 
добавить файл index.html в буфер, и затем сделать commit этого файла в репозиторий


4. В файл index.html добавить 3-й параграф и проверить статус репозитория

5. Закоммитить  файл index.html в репозиторий и проверить статус репозитория

6. Повторить еще раз пункты 4, 5

7.  В каталоге Home_1  создать файл catalog.html
    Проверить статус репозитория - проанализировать сообщение git

8. Добавить файл catalog.html в stage area (буфер) и  опять проверить статус репозитория.
Сравнить сообщение с предыдущим и проанализировать в чем разница

9. Посмотреть историю коммитов
   Если Вы все сделали правильно, то у Вас выведуться 3 лога коммитов
